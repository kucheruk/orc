### Step 4: Merge ALL commits into main
This is the critical integration step. You must merge ALL your work into the main branch:

1. First, get the branch name: `git rev-parse --abbrev-ref HEAD`
2. Switch to main: `git checkout master` (or `git checkout main`)
3. Merge: `git merge --no-ff <your-branch> -m "feat({card_id}): <summary of delivered work>"`
4. If there are conflicts — resolve them. You are the integration expert, this is your job.
5. Verify the merge: `git log --oneline -3` should show your merge commit on main.
6. Switch back to your branch: `git checkout <your-branch>`

**DO NOT skip this step.** If you only commit in the worktree without merging to main, the code will not be delivered.
